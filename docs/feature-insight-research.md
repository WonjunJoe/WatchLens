# WatchLens 기능 인사이트 리서치 & 평가

## 평가 기준

| 기준 | 설명 | 가중치 |
|------|------|--------|
| **Actionability** | 사용자가 이 인사이트를 보고 실제 행동을 바꿀 수 있는가? | 35% |
| **Surprise Factor** | 사용자가 "몰랐던 사실"을 발견하게 하는가? | 30% |
| **Data Availability** | 현재 수집하는 데이터만으로 구현 가능한가? | 20% |
| **Clarity** | 결과가 직관적으로 이해 가능한가? (설명 없이도) | 15% |

점수: 1(낮음) ~ 5(높음), 가중 합산.

## 추가할 신규 기능

### 1. 콘텐츠 다양성 점수 (YouTube)

**근거:** YouTube 알고리즘은 시간이 지날수록 콘텐츠 다양성을 좁힌다 (Brookings 연구). 사용자 85%가 3개 카테고리에 집중되어 있다는 사실을 모른다.

- **Actionability: 5** — "다양한 콘텐츠를 의식적으로 탐색해야겠다"
- **Surprise: 5** — "나는 다양하게 본다고 생각했는데..."
- **Data: 5** — 카테고리 데이터 이미 존재
- **Clarity: 4** — Shannon entropy → 100점 스케일로 변환 필요
- **총점: 4.80** — 최고 우선순위

**구현:** Shannon entropy 계산 → 0~100 스케일 → 월별 트렌드

### 2. 몰아보기(Binge) 세션 감지 (YouTube)

**근거:** 같은 채널/카테고리 연속 시청 세션은 사용자가 인지하지 못하는 가장 큰 시간 소모 원인.

- **Actionability: 5** — "이 채널에서 래빗홀에 빠지지 말자"
- **Surprise: 5** — "3시간 연속으로 봤다고?"
- **Data: 4** — 타임스탬프 기반 세션 클러스터링 필요
- **Clarity: 4** — 세션 정의가 명확해야 함
- **총점: 4.65**

**구현:** 30분 이내 간격 시청 → 하나의 세션, 5+ 연속 = binge, 통계 산출

### 3. 주의력 변화 트렌드 (YouTube)

**근거:** 학술 연구(ERIC, ResearchGate)에서 숏폼 소비와 집중력 저하의 상관관계 입증. 월별 평균 영상 길이 트렌드가 핵심.

- **Actionability: 5** — "의식적으로 긴 영상을 보자"
- **Surprise: 4** — "1월에 12분이던 평균이 지금 3분?"
- **Data: 5** — duration 메타데이터 존재
- **Clarity: 5** — 라인 차트로 직관적
- **총점: 4.75**

**구현:** 월별 평균 영상 길이 + Shorts 비율 트렌드 라인 차트

### 4. 시간 비용 환산 (YouTube)

**근거:** RescueTime/Spotify Wrapped 리서치 — 숫자를 "책 68권" 같은 대체 활동으로 환산하면 행동 변화 유도력이 급증.

- **Actionability: 5** — "이 시간으로 다른 걸 할 수 있었구나"
- **Surprise: 4** — "340시간 = 새 언어 하나?"
- **Data: 5** — 시청 시간 이미 계산됨
- **Clarity: 5** — 비유가 직관적
- **총점: 4.75**

**구현:** 총 시청 시간 → 등가 활동 변환 (책, 영화, 언어 학습, 마라톤 등)

## 구현 우선순위

1. **콘텐츠 다양성 점수** (4.80) — 계산 추가 + 새 컴포넌트
2. **주의력 변화 트렌드** (4.75) — 계산 추가 + 라인 차트
3. **시간 비용 환산** (4.75) — 프레젠테이션 레이어만
4. **몰아보기 세션 감지** (4.65) — 세션 로직 + 새 컴포넌트
